# Changelog

## [1.0.0] - 2026-01-25
### Initial Release
- **Added**: Arrow usage from bundles for bows and crossbows
- **Added**: Support for all bundle color variants
- **Added**: Smart priority system (inventory arrows → bundle arrows)
- **Added**: Infinity enchantment compatibility for normal arrows
- **Added**: Special arrow consumption with Infinity (spectral, tipped)
- **Added**: Bundle content order preservation
- **Added**: Performance optimization with caching system
- **Added**: Support for all arrow types (normal, spectral, tipped, modded)
- **Technical**: Client and server-side compatible
- **Technical**: Optimized iteration reduction (50-65% improvement)
- **Technical**: Cache invalidation system for accuracy
- **Technical**: Built for Minecraft 1.21.11 with Fabric
