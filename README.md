Use arrows stored in bundles with your bow and crossbow! Never run out of arrows in the heat of battle.

## ✨ Features

### 🎯 Smart Arrow Management
- **Bundle Integration** - Use arrows directly from any colored bundle
- **Intelligent Priority** - Automatically uses inventory arrows first, then bundle arrows
- **Infinity Compatible** - Respects Infinity enchantment for normal arrows
- **Special Arrow Support** - Infinity consumes special arrows (spectral, tipped) as intended

### 📦 Bundle Support
- **All Bundle Colors** - Works with white, orange, magenta, and all other bundle variants
- **Order Preservation** - Bundle contents maintain their original order
- **Performance Optimized** - Efficient caching system for minimal performance impact

### ⚔️ Weapon Compatibility
- **Bows** - Full support with all enchantments
- **Crossbows** - Automatically works with crossbows
- **All Arrow Types** - Normal, spectral, tipped, and modded arrows

## 🚀 Installation

1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Install [Fabric API](https://modrinth.com/mod/fabric-api)
3. Download Bundle Arrows mod
4. Place the JAR file in your `.minecraft/mods/` folder

## 🎮 How to Use

1. Place arrows in any bundle
2. Keep the bundle in your inventory
3. Use your bow/crossbow normally
4. Arrows from bundles are used automatically when inventory arrows run out

### 💡 Tips
- Bundle arrows are used **after** inventory arrows are depleted
- Infinity enchantment works correctly (doesn't consume normal arrows)
- Special arrows (spectral, tipped) are always consumed, even with Infinity
- Works in both singleplayer and multiplayer

## 🔧 Technical Details
